What is cloud computing?

the cloud computing is hosting your servers in the cloud instead of on-premise,it also refers the using the right resources at the right time.

there are several Cloud Service Providers(CSP) like Microsoft Azure, Amazon Web Service(AWS) and Google Cloud platform

Microsoft Azure:

Azure is continually expanding set of cloud Services that help your organization to meet with your current and future challenges.

What does Azure offers?

  • Be ready for the future.
  • Build on your own terms.
  • Trust on your cloud.

Azure portal

is a web-based, unified console that provides an alteranive to the command line tools.

Azure Services

the basic azure services are:

  • Compute: hosting applications and services.
  • Networking: provides access to applications.
  • Mobile.
  • Storage: provides resources like file storage, database storage,...etc.
  • Database: store wide variety of databases.
  • Web.
  • Big Data.
  • IoT.
  • AI.
  • DevOps

Azure Cloud Key Characteristics

Scalability

The ability to Allocate/DeAllocate(add/remove) devices.

Elasticity

The ability to Allocate/remove devices dynamically.

Agility

The ability to allocate and de-allocate Quickly.

Fault Tolerance

The abilty to remain up and running the system during component and service failure.

Disaster Recovery

The ability to the system to recover from an event that has taken down the service ,so your system will work properly after the disaster.

High availability

The ability to keep services running for extended period of time with a very short downtime
